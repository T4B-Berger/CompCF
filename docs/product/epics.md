## MVP / Post-MVP / Out-of-Scope Conventions
- **MVP:** smallest end-to-end workflow that can run a real CrossFit competition safely.
- **Post-MVP:** optimization, automation depth, reporting sophistication.
- **Out-of-Scope:** non-CrossFit workflows, enterprise-only customization, non-GitHub planning systems.
